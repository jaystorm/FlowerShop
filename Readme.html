FlowerShop Hands On Lab

This is an example of to build a simple EShop app for a mobile device/HTML5 capable client with JayStorm and JayData.

The Shop and Order data is stored in a JayStorm application hosted at JayStack.com. You are encouraged to build you own shop database based on this example. To set up your own cloud backed JayData endpoint read this: http://jaystack.com/blog/creating-your-first-javascript-data-service

This code builds on knockoutJS to handle the client application aspects. You can of course use your own favorite client library since the data layer is JayData and is not coupled with knockoutJS.

Study the index.html file in each step. Starting from the first each level adds a bit of functionality.

To run the code simply open the index.html page found in every folder in any browser but IE9- (IE10 is OK). The folder structure can be served by a web browser or can be accessed directly from the disk as a file:// resource. Note that JayStorm uses CORS for universal cross origin access and IE9- does not support CORS as per the standard.
